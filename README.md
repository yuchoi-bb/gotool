# gotool

즐겨찾기(바로가기) 런처. 실행하면 마우스 커서 위치에 우클릭 메뉴 모양의 **4열 팝업**이 뜹니다.

```
🌐 웹   |   💻 개발   |   📦 비개발   |   🐿 다람쥐
```

- **왼쪽 클릭** → 해당 항목 실행(웹은 브라우저, 앱은 실행, 폴더는 탐색기)
- **드래그 & 드롭** → 항목을 마우스로 끌어 다른 열에 놓으면 이동. 창은 닫히지 않고 제자리에 유지
- **오른쪽 클릭** → 삭제(확인창)
- **`!` 표시(이름 오른쪽)** → 추가된 지 10분 이내의 신규 항목
- **⬆ 새 버전 다운로드** → 새 릴리스가 나오면 맨 위에 다운로드 버튼이 자동 표시
- **ESC 또는 바깥 클릭** → 닫기

## 자동 분류

`shortcuts` 폴더에 섞어 넣어도 자동으로 분류됩니다.

| 항목 | 분류 |
|---|---|
| `.url` (http/https 주소) | 🌐 웹 |
| 이름·대상 경로에 개발 키워드(code, git, docker, python, sql, terminal 등) | 💻 개발 |
| 그 외 앱/파일/폴더 | 📦 비개발 |
| 🐿 다람쥐 | 수동 이동 전용 |

분류가 마음에 안 들면 **항목을 마우스로 끌어서 원하는 열에 놓으면** 됩니다.

## 수동 이동 (드래그 & 드롭)

- **패널에서**: 항목을 누른 채 원하는 열 위로 끌어다 놓으면 이동합니다. 이동해도 창은 닫히지 않고 같은 위치에 유지됩니다.
- **탐색기에서**: `shortcuts` 폴더 안의 `웹`, `개발`, `비개발`, `다람쥐` 하위 폴더로 파일을 드래그해도 됩니다.
  (하위 폴더에 있으면 그 열로 고정, 루트에 있는 파일만 자동 분류 대상)

## 바로가기 추가

바로가기는 `gotool.exe` 옆의 **`shortcuts` 폴더**(첫 실행 시 자동 생성)에 파일로 저장됩니다.

1. **탐색기 우클릭(추천)** — 한 번만 `gotool.exe install` 실행(또는 메뉴 하단의 등록 항목 클릭)하면
   파일/폴더 우클릭 메뉴에 **"gotool에 추가"**가 생깁니다. (Windows 11은 "더 많은 옵션 표시" 안)
   - 웹: 브라우저 주소창의 자물쇠를 바탕화면에 드래그해 `.url`을 만든 뒤 우클릭 → "gotool에 추가"
2. **폴더에 직접 넣기** — `.lnk`/`.url`/파일을 `shortcuts` 폴더(또는 카테고리 하위 폴더)에 복사
3. **명령행** — `gotool.exe add "C:\경로"`

새로 추가된 항목은 10분 동안 이름 오른쪽에 `!`가 붙습니다.

## 수정 · 삭제

- **삭제**: 항목 우클릭 → 확인
- **수정(이름 변경 등)**: 메뉴 맨 아래 **"⚙ 바로가기 폴더 열기"** → 파일 이름을 바꾸면 메뉴 이름이 바뀝니다
- 우클릭 메뉴 등록 해제: `gotool.exe uninstall`

## 자동 업데이트 알림

실행할 때 GitHub 최신 릴리스를 확인해서(약 0.7초 대기, 오프라인이면 건너뜀)
새 버전이 있으면 메뉴 맨 위에 **"⬆ 새 버전 vX.Y.Z 다운로드"** 버튼이 나타납니다.
클릭하면 브라우저로 새 `gotool.exe`가 다운로드됩니다. 받은 파일로 기존 exe를 덮어쓰면 됩니다.

## AutoHotkey 연동 (ESC 두 번으로 실행)

```ahk
~Esc::
if (A_PriorHotkey = "~Esc" && A_TimeSincePriorHotkey < 400)
    Run "C:\tools\gotool.exe"
return
```

## 명령어 정리

| 명령 | 동작 |
|---|---|
| `gotool.exe` | 4열 메뉴 표시 (exe 옆 `shortcuts` 폴더) |
| `gotool.exe <폴더>` | 지정 폴더의 내용을 메뉴로 표시 |
| `gotool.exe add <경로>` | 바로가기 추가 (`.lnk`/`.url`은 복사, 그 외는 바로가기 생성) |
| `gotool.exe install` | 탐색기 우클릭 "gotool에 추가" 등록 |
| `gotool.exe uninstall` | 우클릭 메뉴 등록 해제 |

## 다운로드 / 빌드

[Releases](../../releases/latest) 페이지에서 `gotool.exe`를 내려받으세요.

```
go build -trimpath -ldflags "-s -w -H windowsgui -X main.version=v0.4.0" -o gotool.exe .
```

`main`에 푸시하거나 `v*` 태그를 만들면 GitHub Actions가 자동으로 빌드해 릴리스에 `gotool.exe`를 올립니다.
